# House-Price-Prediction
A Machine Learning project that predicts house prices based on features like area, bedrooms, parking, and amenities using Linear Regression and Streamlit.
# 🏠 House Price Prediction System

This project predicts Indian house prices using Machine Learning.

## Technologies Used

* Python
* Pandas
* Scikit-learn
* Streamlit

## Features

* Area-based prediction
* Bedroom prediction
* Car parking option
* Swimming pool option
* Interactive Streamlit UI

## Machine Learning Model

Linear Regression

## How to Run

```bash
pip install -r requirements.txt
streamlit run app.py
```
